### Проектные записи ядра  
- Бенчмарк при старте.  
  Определение радиуса загрузки чанков. (мощнее - больше)  
  Определение множителя длительности такта. (мощнее - меньше)  
- Система чанков с хранением в виде сжатых данных при отгрузке.  
- Радиус загрузки чанков с распаковкой при подгрузке.  
- Грубая симуляция с низкой частотой.  
  Производится один раз в такт.  
- Система тактов.  
  Длительность такта определяется количеством неактивных сущностей.  
  Длительность такта должна быть такой, чтобы ФПС был стабилен при динамически изменяющейся длительности.
